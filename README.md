# RecipeManager

This is an app that manages recipes for you and your family. You can share your recipes to everyone if you like, or view the public repository of thousands of recipes for free!

### Running the app locally

* Frontend - run the following in the frontend directory
    - `npm install`
    - `docker build -t recipemanager:latest .`
    - `npm run docker`

* Backend - in visual studio, set your debugger to dockerfile. Run.
