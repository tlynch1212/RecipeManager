## Intro 

Front end for the Recipe Manager

## Usage 

* Docker
    - Build image - `npm build docker`
    - Run Container - `npm run docker`
* Npm
    - node - `npm install`
    - build - `npm build`
    - test - `npm test`
    - run - `npm start`
